#### 실습 1-1: 자기소개 프로그램 (15분)

**목표:** `System.out.println`으로 출력하고, 변수에 값을 저장하는 법을 익힌다.

**단계별 가이드:**

1단계: 파일 만들기
- 파일 이름은 반드시 `Intro.java` (클래스 이름과 동일)
- 모든 Java 프로그램은 `public class 클래스이름 { }` 안에 작성

2단계: main 메서드 작성
- Java 프로그램의 시작점은 항상 `public static void main(String[] args)`
- 이 안에 실행할 코드를 넣는다

3단계: 변수 선언과 출력
- `String name = "홍길동";` → 문자열 변수
- `int age = 17;` → 정수 변수
- `System.out.println(name);` → 변수 값 출력

4단계: 문자열 연결
- `"이름: " + name` → 문자열과 변수를 `+`로 연결

**완성 코드:**

```java
public class Intro {

        System.out.println("학교: " + school);
        System.out.println("취미: " + hobby);
        System.out.println("================");
    }
}
```

#### 실습 1-2: 간단한 계산기 (20분)

**목표:** 정수/실수 변수, 사칙연산, 나머지 연산을 연습한다.

**단계별 가이드:**

1단계: 두 정수 변수 선언
- `int a = 17;`과 `int b = 5;` 선언

2단계: 사칙연산 결과 출력
- `+`, `-`, `*`, `/` (정수 나눗셈!), `%` (나머지)

3단계: 정수 나눗셈 vs 실수 나눗셈 차이 확인
- `17 / 5` → `3` (소수점 버림!)
- `17.0 / 5` → `3.4` (하나라도 실수면 실수 결과)

4단계: Scanner로 입력받기 (새로운 도구 3가지를 배웁니다!)
- **import**: Java에 기본으로 포함되지 않은 기능을 가져오는 것 (도서관에서 책을 빌려오는 것과 같음)
  - `import java.util.Scanner;` → Scanner라는 도구를 가져옴
- **Scanner**: 키보드에서 사용자의 입력을 받을 수 있게 해주는 도구
  - `Scanner sc = new Scanner(System.in);` → Scanner 도구를 만듦
  - **new**: "새로운 도구를 만들어라"라는 명령
- **sc.nextInt()**: 사용자가 타이핑한 정수를 읽어옴

**완성 코드:**

```java
import java.util.Scanner;

public class Calculator {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // 사용자 입력
        System.out.print("첫 번째 정수: ");
        int a = sc.nextInt();
        System.out.print("두 번째 정수: ");
        int b = sc.nextInt();

        // 사칙연산
        System.out.println(a + " + " + b + " = " + (a + b));
        System.out.println(a + " - " + b + " = " + (a - b));
        System.out.println(a + " * " + b + " = " + (a * b));
        System.out.println(a + " / " + b + " = " + (a / b) + " (정수 나눗셈)");
        System.out.println(a + " % " + b + " = " + (a % b) + " (나머지)");

        // 실수 나눗셈
        // (double)은 "이 숫자를 잠시 실수(소수점 있는 수)로 취급해줘"라는 뜻 (캐스팅)
        double result = (double) a / b;
        System.out.println(a + " / " + b + " = " + result + " (실수 나눗셈)");

        sc.close();  // Scanner 사용이 끝나면 정리 (사용 후 도구를 제자리에 놓는 것)
    }
}
```

#### 숙제 1-1: 온도 변환기 [기초]

화씨(F)를 섭씨(C)로, 섭씨를 화씨로 변환하는 프로그램을 작성하세요.

- 공식: `C = (F - 32) * 5 / 9`, `F = C * 9 / 5 + 32`
- 섭씨 25도를 화씨로, 화씨 98.6도를 섭씨로 변환하여 출력

**완성 코드:**

```java
public class TempConverter {
    public static void main(String[] args) {
        // 섭씨 → 화씨
        double celsius = 25.0;
        double fahrenheit = celsius * 9.0 / 5 + 32;
        System.out.println(celsius + "°C = " + fahrenheit + "°F");

        // 화씨 → 섭씨
        double fahr = 98.6;
        double cel = (fahr - 32) * 5.0 / 9;
        System.out.println(fahr + "°F = " + cel + "°C");
    }
}
```

#### 실습 2-1: 이름 포맷터 (15분)

**목표:** String 메서드(`length`, `substring`, `toUpperCase`, `indexOf` 등)를 익힌다.

**단계별 가이드:**

1단계: String 변수에 전체 이름 저장
- `String fullName = "kim minjun";`

2단계: 주요 String 메서드 실험
- `.length()` → 문자열 길이
- `.toUpperCase()` / `.toLowerCase()` → 대소문자 변환
- `.substring(시작, 끝)` → 부분 문자열 (끝 인덱스는 미포함!)
- `.indexOf(" ")` → 공백 위치 찾기

3단계: 성과 이름 분리
- 공백 위치를 찾아서 `substring`으로 분리

4단계: "Last, First" 형식으로 조합

**완성 코드:**

```java
public class NameFormatter {
    public static void main(String[] args) {
        String fullName = "kim minjun";

        // 기본 메서드
        System.out.println("길이: " + fullName.length());
        System.out.println("대문자: " + fullName.toUpperCase());
        System.out.println("소문자: " + fullName.toLowerCase());

        // 성과 이름 분리
        int spaceIndex = fullName.indexOf(" ");
        String lastName = fullName.substring(0, spaceIndex);
        String firstName = fullName.substring(spaceIndex + 1);

        // 첫 글자만 대문자로
        lastName = lastName.substring(0, 1).toUpperCase() + lastName.substring(1);
        firstName = firstName.substring(0, 1).toUpperCase() + firstName.substring(1);

        System.out.println("포맷: " + lastName + ", " + firstName);
        System.out.println("이니셜: " + lastName.charAt(0) + "." + firstName.charAt(0) + ".");
    }
}
```

**단계별 가이드:**

1단계: Math.random() 이해
- `Math.random()` → 0.0 이상 1.0 미만의 실수
- `(int)(Math.random() * 6)` → 0~5 정수
- `(int)(Math.random() * 6) + 1` → 1~6 정수 (주사위!)

2단계: 주사위 2개 굴리기
- 각각 변수에 저장, 합 계산

3단계: 결과 판정
- 합이 7이면 "럭키 세븐!" 출력

**완성 코드:**

```java
public class DiceGame {
    public static void main(String[] args) {
        // 주사위 2개 굴리기
        int die1 = (int)(Math.random() * 6) + 1;
        int die2 = (int)(Math.random() * 6) + 1;

        System.out.println("주사위 1: " + die1);
        System.out.println("주사위 2: " + die2);

        System.out.println("Math.max(3, 7) = " + Math.max(3, 7));
```

**단계별 가이드:**

1단계: 점수 입력받기
- `Scanner`로 정수 입력

2단계: 등급 기준 정하기
- 90 이상: A, 80 이상: B, 70 이상: C, 60 이상: D, 그 외: F
- `if-else if` 순서가 중요! (높은 점수부터)

3단계: 경계값 테스트
- 90, 89, 0, 100 등으로 테스트

**완성 코드:**

```java
import java.util.Scanner;

public class GradeChecker {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        System.out.print("점수 입력 (0-100): ");
        int score = sc.nextInt();

        String grade;

        if (score >= 90) {
            grade = "A";
        } else if (score >= 80) {
            grade = "B";
        } else if (score >= 70) {
            grade = "C";
        } else if (score >= 60) {
            grade = "D";
        } else {
            grade = "F";
        }

        System.out.println("점수: " + score + " → 등급: " + grade);

        // +/- 세분화
        if (!grade.equals("F")) {
            int mod = score % 10;
            if (mod >= 7) {
                grade += "+";
            } else if (mod < 3) {
                grade += "-";
            }
        }
        System.out.println("세분화 등급: " + grade);

        sc.close();
    }
}
```

#### 실습 3-2: 윤년 판별기 (20분)

**목표:** 논리 연산자 `&&`, `||`를 활용한 복합 조건문을 작성한다.

**단계별 가이드:**

1단계: 윤년 규칙 이해
- 4로 나누어 떨어지면 윤년
- 단, 100으로 나누어 떨어지면 윤년 아님
- 단, 400으로 나누어 떨어지면 윤년
- 정리: `(4의 배수 && 100의 배수 아님) || 400의 배수`

2단계: 조건문 작성
- `year % 4 == 0` → 4의 배수
- `&&`, `||` 사용

3단계: 테스트
- 2024(윤년), 1900(평년), 2000(윤년), 2023(평년)

**완성 코드:**

```java
import java.util.Scanner;

public class LeapYear {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        System.out.print("년도 입력: ");
        int year = sc.nextInt();

        boolean isLeap = (year % 4 == 0 && year % 100 != 0) || (year % 400 == 0);

        if (isLeap) {
            System.out.println(year + "년은 윤년입니다. (2월 29일)");
        } else {
            System.out.println(year + "년은 평년입니다. (2월 28일)");
        }

        // 테스트 케이스 자동 확인
        System.out.println("\n=== 테스트 ===");
        int[] testYears = {2024, 1900, 2000, 2023, 1600, 1700};
        for (int y : testYears) {
            boolean leap = (y % 4 == 0 && y % 100 != 0) || (y % 400 == 0);
            System.out.println(y + ": " + (leap ? "윤년" : "평년"));
        }

        sc.close();
    }
}
```

### 숙제 (40분)

#### 숙제 3-1: BMI 계산기 [기초]

키(cm)와 몸무게(kg)를 입력받아 BMI를 계산하고 판정하세요.

- BMI = 몸무게 / (키(m))^2
- 18.5 미만: 저체중, 18.5~24.9: 정상, 25~29.9: 과체중, 30 이상: 비만

**완성 코드:**

```java
import java.util.Scanner;

public class BMI {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        System.out.print("키(cm): ");
        double heightCm = sc.nextDouble();
        System.out.print("몸무게(kg): ");
        double weight = sc.nextDouble();

        double heightM = heightCm / 100.0;
        double bmi = weight / (heightM * heightM);

        System.out.printf("BMI: %.1f → ", bmi);

        if (bmi < 18.5) {
            System.out.println("저체중");
        } else if (bmi < 25.0) {
            System.out.println("정상");
        } else if (bmi < 30.0) {
            System.out.println("과체중");
        } else {
            System.out.println("비만");
        }

        sc.close();
    }
}
```

#### 숙제 5-3: Fraction 클래스 [도전]

분수(분자/분모)를 표현하는 Fraction 클래스를 만드세요.
- `add(Fraction other)` → 두 분수의 합 반환
- `multiply(Fraction other)` → 두 분수의 곱 반환
- 결과는 기약분수로 (GCD 활용)

**완성 코드:**

```java
public class Fraction {
    private int numerator;
    private int denominator;

    public Fraction(int num, int den) {
        if (den == 0) {
            throw new IllegalArgumentException("분모는 0일 수 없습니다.");
        }
        // 음수 처리: 분모가 음수면 부호를 분자로 이동
        if (den < 0) {
            num = -num;
            den = -den;
        }
        int g = gcd(Math.abs(num), den);
        this.numerator = num / g;
        this.denominator = den / g;
    }

    private static int gcd(int a, int b) {
        while (b != 0) {
            int temp = b;
            b = a % b;
            a = temp;
        }
        return a;
    }

    public Fraction add(Fraction other) {
        int num = this.numerator * other.denominator + other.numerator * this.denominator;
        int den = this.denominator * other.denominator;
        return new Fraction(num, den);
    }

    public Fraction multiply(Fraction other) {
        return new Fraction(this.numerator * other.numerator,
                            this.denominator * other.denominator);
    }

    public String toString() {
        if (denominator == 1) return "" + numerator;
        return numerator + "/" + denominator;
    }

    public static void main(String[] args) {
        Fraction f1 = new Fraction(1, 3);
        Fraction f2 = new Fraction(2, 5);

        System.out.println(f1 + " + " + f2 + " = " + f1.add(f2));
        System.out.println(f1 + " * " + f2 + " = " + f1.multiply(f2));

        Fraction f3 = new Fraction(3, 4);
        Fraction f4 = new Fraction(1, 6);
        System.out.println(f3 + " + " + f4 + " = " + f3.add(f4));
        System.out.println(f3 + " * " + f4 + " = " + f3.multiply(f4));
    }
}
```

```java
    public static void main(String[] args) {
        int[] arr = {2, 5, 8, 12, 16, 23, 38, 56, 72, 91};
        int target = 23;

        System.out.print("배열: ");
        for (int v : arr) System.out.print(v + " ");
        System.out.println("\n찾을 값: " + target);

        int low = 0;
        int high = arr.length - 1;
        int steps = 0;
        int result = -1;

        while (low <= high) {
            steps++;
            int mid = (low + high) / 2;
            System.out.println("  단계 " + steps + ": arr[" + mid + "] = " + arr[mid]);

            if (arr[mid] == target) {
                result = mid;
                break;
            } else if (arr[mid] < target) {
                low = mid + 1;
            } else {
                high = mid - 1;
            }
        }

        if (result != -1) {
            System.out.println("찾았습니다! 인덱스: " + result + " (" + steps + "단계)");
        } else {
            System.out.println("찾지 못했습니다.");
        }
    }
```

```java
import java.util.Scanner;

public class TicTacToe {
    static char[][] board = new char[3][3];

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // 보드 초기화
        for (int r = 0; r < 3; r++) {
            for (int c = 0; c < 3; c++) {
                board[r][c] = ' ';
            }
        }

        char currentPlayer = 'X';
        int moves = 0;

        while (moves < 9) {
            printBoard();
            System.out.println("플레이어 " + currentPlayer + "의 차례");

            int row, col;
            while (true) {
                System.out.print("행(0-2): ");
                row = sc.nextInt();
                System.out.print("열(0-2): ");
                col = sc.nextInt();

                if (row >= 0 && row < 3 && col >= 0 && col < 3 && board[row][col] == ' ') {
                    break;
                }
                System.out.println("잘못된 위치입니다. 다시 입력하세요.");
            }

            board[row][col] = currentPlayer;
            moves++;

            if (checkWin(currentPlayer)) {
                printBoard();
                System.out.println("플레이어 " + currentPlayer + " 승리!");
                sc.close();
                return;
            }

            // 플레이어 교대
            currentPlayer = (currentPlayer == 'X') ? 'O' : 'X';
        }

        printBoard();
        System.out.println("무승부!");
        sc.close();
    }

    static void printBoard() {
        System.out.println("\n  0 1 2");
        for (int r = 0; r < 3; r++) {
            System.out.print(r + " ");
            for (int c = 0; c < 3; c++) {
                System.out.print(board[r][c]);
                if (c < 2) System.out.print("|");
            }
            System.out.println();
            if (r < 2) System.out.println("  -----");
        }
        System.out.println();
    }

    static boolean checkWin(char p) {
        // 가로
        for (int r = 0; r < 3; r++) {
            if (board[r][0] == p && board[r][1] == p && board[r][2] == p) return true;
        }
        // 세로
        for (int c = 0; c < 3; c++) {
            if (board[0][c] == p && board[1][c] == p && board[2][c] == p) return true;
        }
        // 대각선
        if (board[0][0] == p && board[1][1] == p && board[2][2] == p) return true;
        if (board[0][2] == p && board[1][1] == p && board[2][0] == p) return true;

        return false;
    }
```

3x3 배열이 마방진인지 검사하세요. (모든 행, 열, 대각선의 합이 같으면 마방진)

**완성 코드:**

```java
public class MagicSquare {
    public static void main(String[] args) {
        int[][] square = {
            {2, 7, 6},
            {9, 5, 1},
            {4, 3, 8}
        };

        // 출력
        System.out.println("=== 마방진 검증 ===");
        for (int r = 0; r < 3; r++) {
            for (int c = 0; c < 3; c++) {
                System.out.print(square[r][c] + " ");
            }
            System.out.println();
        }

        boolean isMagic = true;
        int targetSum = square[0][0] + square[0][1] + square[0][2]; // 첫 행의 합

        // 행 검사
        for (int r = 0; r < 3; r++) {
            int rowSum = 0;
            for (int c = 0; c < 3; c++) {
                rowSum += square[r][c];
            }
            if (rowSum != targetSum) {
                isMagic = false;
                System.out.println("행 " + r + " 합: " + rowSum + " (불일치!)");
            }
        }

        // 열 검사
        for (int c = 0; c < 3; c++) {
            int colSum = 0;
            for (int r = 0; r < 3; r++) {
                colSum += square[r][c];
            }
            if (colSum != targetSum) {
                isMagic = false;
                System.out.println("열 " + c + " 합: " + colSum + " (불일치!)");
            }
        }

        // 대각선 검사
        int diag1 = square[0][0] + square[1][1] + square[2][2];
        int diag2 = square[0][2] + square[1][1] + square[2][0];
        if (diag1 != targetSum || diag2 != targetSum) {
            isMagic = false;
        }

        System.out.println("목표 합: " + targetSum);
        System.out.println(isMagic ? "마방진입니다!" : "마방진이 아닙니다.");
    }
}
```

두 행렬의 덧셈과 곱셈을 수행하세요.

**완성 코드:**

```java
public class MatrixOps {
    public static void main(String[] args) {
        int[][] A = {{1, 2}, {3, 4}};
        int[][] B = {{5, 6}, {7, 8}};

        // 행렬 덧셈
        System.out.println("=== A + B ===");
        int[][] sum = new int[2][2];
        for (int r = 0; r < 2; r++) {
            for (int c = 0; c < 2; c++) {
                sum[r][c] = A[r][c] + B[r][c];
            }
        }
        printMatrix(sum);

        // 행렬 곱셈
        System.out.println("=== A * B ===");
        int[][] product = new int[2][2];
        for (int r = 0; r < 2; r++) {
            for (int c = 0; c < 2; c++) {
                product[r][c] = 0;
                for (int k = 0; k < 2; k++) {
                    product[r][c] += A[r][k] * B[k][c];
                }
            }
        }
        printMatrix(product);
    }

    static void printMatrix(int[][] m) {
        for (int r = 0; r < m.length; r++) {
            for (int c = 0; c < m[r].length; c++) {
                System.out.printf("%4d", m[r][c]);
            }
            System.out.println();
        }
        System.out.println();
    }
}
```

#### 숙제 8-3: 지뢰찾기 보드 생성 [도전]

N x N 보드에 M개의 지뢰를 랜덤 배치하고, 각 칸에 주변 지뢰 개수를 표시하세요.

**완성 코드:**

```java
public class Minesweeper {
    public static void main(String[] args) {
        int N = 8;  // 보드 크기
        int M = 10; // 지뢰 수

        boolean[][] mines = new boolean[N][N];
        int[][] board = new int[N][N];

        // 지뢰 랜덤 배치
        int placed = 0;
        while (placed < M) {
            int r = (int)(Math.random() * N);
            int c = (int)(Math.random() * N);
            if (!mines[r][c]) {
                mines[r][c] = true;
                placed++;
            }
        }

        // 주변 지뢰 수 계산
        for (int r = 0; r < N; r++) {
            for (int c = 0; c < N; c++) {
                if (mines[r][c]) {
                    board[r][c] = -1; // 지뢰
                    continue;
                }
                int count = 0;
                // 8방향 탐색
                for (int dr = -1; dr <= 1; dr++) {
                    for (int dc = -1; dc <= 1; dc++) {
                        int nr = r + dr;
                        int nc = c + dc;
                        if (nr >= 0 && nr < N && nc >= 0 && nc < N && mines[nr][nc]) {
                            count++;
                        }
                    }
                }
                board[r][c] = count;
            }
        }

        // 정답 보드 출력
        System.out.println("=== 지뢰찾기 보드 (" + N + "x" + N + ", 지뢰 " + M + "개) ===");
        System.out.print("  ");
        for (int c = 0; c < N; c++) System.out.print(c + " ");
        System.out.println();

        for (int r = 0; r < N; r++) {
            System.out.print(r + " ");
            for (int c = 0; c < N; c++) {
                if (board[r][c] == -1) {
                    System.out.print("* ");
                } else if (board[r][c] == 0) {
                    System.out.print(". ");
                } else {
                    System.out.print(board[r][c] + " ");
                }
            }
            System.out.println();
        }
    }
}
```

**완성 코드:**

```java
public class WordScramble {

    public static String reverseSubstring(String str, int start, int end) {
        String before = str.substring(0, start);
        String middle = str.substring(start, end);
        String after = str.substring(end);

        // middle 뒤집기
        String reversed = "";
        for (int i = middle.length() - 1; i >= 0; i--) {
            reversed += middle.substring(i, i + 1);
        }

        return before + reversed + after;
    }

    public static String scramble(String str, String target) {
        String result = str;
        int idx = result.indexOf(target);

        while (idx != -1) {
            result = reverseSubstring(result, idx, idx + target.length());
            idx = result.indexOf(target, idx + target.length());
        }

        return result;
    }

    public static void main(String[] args) {
        // 테스트
        System.out.println(reverseSubstring("abcdefg", 2, 5));
        // "abedcfg" (cde → edc)

        System.out.println(scramble("hello world hello", "hello"));
        // "olleh world olleh"

        System.out.println(scramble("abcabcabc", "abc"));
        // "cbacbacba"  -- 3개의 "abc"가 각각 독립적으로 뒤집힘 (idx가 target.length()씩 건너뜀)
    }
}
```

#### FRQ 2: ScoreAnalyzer (ArrayList 알고리즘)

학생 점수 목록(`ArrayList<Integer>`)을 분석하는 `ScoreAnalyzer` 클래스를 구현하세요.

(a) `removeBelow` 메서드: 주어진 기준 점수 미만인 점수를 모두 제거하고, 제거된 점수의 개수를 반환  
(b) `getScoreFrequency` 메서드: 특정 점수가 목록에 몇 번 나타나는지 반환하고, 해당 점수를 모두 목록 앞쪽으로 이동(원래 순서 유지)

**완성 코드:**

```java
import java.util.ArrayList;

public class ScoreAnalyzer {
    private ArrayList<Integer> scores;

    public ScoreAnalyzer(ArrayList<Integer> scores) {
        this.scores = scores;
    }

    // (a) 기준 미만 점수 제거, 제거 개수 반환
    public int removeBelow(int threshold) {
        int count = 0;
        int i = 0;
        while (i < scores.size()) {
            if (scores.get(i) < threshold) {
                scores.remove(i);
                count++;
            } else {
                i++;
            }
        }
        return count;
    }

    // (b) 특정 점수의 빈도수 반환 + 해당 점수를 앞쪽으로 이동
    public int getScoreFrequency(int target) {
        ArrayList<Integer> matched = new ArrayList<Integer>();
        ArrayList<Integer> others = new ArrayList<Integer>();

        for (int i = 0; i < scores.size(); i++) {
            if (scores.get(i) == target) {
                matched.add(scores.get(i));
            } else {
                others.add(scores.get(i));
            }
        }

        scores.clear();
        for (int i = 0; i < matched.size(); i++) {
            scores.add(matched.get(i));
        }
        for (int i = 0; i < others.size(); i++) {
            scores.add(others.get(i));
        }

        return matched.size();
    }

    public ArrayList<Integer> getScores() {
        return scores;
    }

    public static void main(String[] args) {
        // (a) 테스트
        ArrayList<Integer> list1 = new ArrayList<Integer>();
        list1.add(85); list1.add(42); list1.add(91); list1.add(55); list1.add(30);
        ScoreAnalyzer sa1 = new ScoreAnalyzer(list1);
        System.out.println("Removed: " + sa1.removeBelow(50));  // 2 (42, 30 제거)
        System.out.println("After removeBelow: " + sa1.getScores()); // [85, 91, 55]

        // (b) 테스트
        ArrayList<Integer> list2 = new ArrayList<Integer>();
        list2.add(70); list2.add(85); list2.add(90); list2.add(85); list2.add(60);
        ScoreAnalyzer sa2 = new ScoreAnalyzer(list2);
        System.out.println("\nFrequency of 85: " + sa2.getScoreFrequency(85)); // 2
        System.out.println("After getScoreFrequency: " + sa2.getScores());     // [85, 85, 70, 90, 60]
    }
}
```

#### FRQ 3: StringFormatter (문자열 처리)

단어 목록과 최대 줄 폭(maxWidth)이 주어질 때, 각 줄에 단어를 채우되 maxWidth를 초과하지 않도록 포맷팅하세요.

(a) `formatLine`: 단어 배열과 시작/끝 인덱스를 받아 한 줄 문자열 반환 (단어 사이 공백 1개)  
(b) `formatAll`: 전체 단어 목록을 줄 단위로 나눠 ArrayList<String> 반환

**완성 코드:**

```java
import java.util.ArrayList;

public class StringFormatter {

    // (a) words[start]부터 words[end] 직전까지를 한 줄로 연결
    public static String formatLine(String[] words, int start, int end) {
        String line = "";
        for (int i = start; i < end; i++) {
            if (i > start) {
                line += " ";
            }
            line += words[i];
        }
        return line;
    }

    // (b) 단어들을 maxWidth 이내로 줄 나누기
    public static ArrayList<String> formatAll(String[] words, int maxWidth) {
        ArrayList<String> lines = new ArrayList<String>();
        int i = 0;

        while (i < words.length) {
            int lineStart = i;
            int lineLength = words[i].length();
            i++;

            while (i < words.length && lineLength + 1 + words[i].length() <= maxWidth) {
                lineLength += 1 + words[i].length();
                i++;
            }

            lines.add(formatLine(words, lineStart, i));
        }

        return lines;
    }

    public static void main(String[] args) {
        String[] words = {"AP", "Computer", "Science", "A", "is", "a",
                          "great", "course", "for", "learning", "Java",
                          "programming"};
        int maxWidth = 20;

        System.out.println("Max width: " + maxWidth);
        System.out.println("--------------------");

        ArrayList<String> lines = formatAll(words, maxWidth);
        for (int i = 0; i < lines.size(); i++) {
            String line = lines.get(i);
            System.out.println("|" + line + "|" + " (길이: " + line.length() + ")");
        }
    }
}
```

#### 프로젝트: 학교 시험 성적 분석 시스템

여러 과목의 시험 성적 데이터를 분석하는 프로그램. 수업 시간에 기본 틀을 함께 만들고, 숙제로 기능을 확장한다.

**완성 코드:**

```java
import java.util.ArrayList;

public class ExamAnalyzer {

    // 학생 클래스
    static class Student {
        private String name;
        private int[] scores; // 과목별 점수
        private String[] subjects;

        public Student(String name, String[] subjects, int[] scores) {
            this.name = name;
            this.subjects = subjects;
            this.scores = scores;
        }

        public String getName() { return name; }
        public int[] getScores() { return scores; }
        public String[] getSubjects() { return subjects; }

        public double getAverage() {
            int sum = 0;
            for (int s : scores) sum += s;
            return (double) sum / scores.length;
        }

        public int getHighest() {
            int max = scores[0];
            for (int s : scores) {
                if (s > max) max = s;
            }
            return max;
        }

        public int getLowest() {
            int min = scores[0];
            for (int s : scores) {
                if (s < min) min = s;
            }
            return min;
        }

        public String getGrade() {
            double avg = getAverage();
            if (avg >= 90) return "A";
            if (avg >= 80) return "B";
            if (avg >= 70) return "C";
            if (avg >= 60) return "D";
            return "F";
        }
    }

    // === 분석 메서드들 ===

    // 전체 학생 성적표 출력
    public static void printReport(ArrayList<Student> students) {
        String[] subjects = students.get(0).getSubjects();

        // 헤더
        System.out.printf("%-8s", "이름");
        for (String s : subjects) {
            System.out.printf("%8s", s);
        }
        System.out.printf("%8s%6s\n", "평균", "등급");
        System.out.println("=".repeat(8 + subjects.length * 8 + 14));

        // 학생별 데이터
        for (int i = 0; i < students.size(); i++) {
            Student st = students.get(i);
            System.out.printf("%-8s", st.getName());
            for (int s : st.getScores()) {
                System.out.printf("%8d", s);
            }
            System.out.printf("%8.1f%6s\n", st.getAverage(), st.getGrade());
        }
    }

    // 과목별 평균 계산
    public static double[] getSubjectAverages(ArrayList<Student> students) {
        int numSubjects = students.get(0).getScores().length;
        double[] averages = new double[numSubjects];

        for (int s = 0; s < numSubjects; s++) {
            int sum = 0;
            for (int i = 0; i < students.size(); i++) {
                sum += students.get(i).getScores()[s];
            }
            averages[s] = (double) sum / students.size();
        }
        return averages;
    }

    // 1등 찾기
    public static Student findTopStudent(ArrayList<Student> students) {
        Student top = students.get(0);
        for (int i = 1; i < students.size(); i++) {
            if (students.get(i).getAverage() > top.getAverage()) {
                top = students.get(i);
            }
        }
        return top;
    }

    // 등급 분포
    public static void printGradeDistribution(ArrayList<Student> students) {
        int[] counts = new int[5]; // A, B, C, D, F
        for (int i = 0; i < students.size(); i++) {
            String g = students.get(i).getGrade();
            if (g.equals("A")) counts[0]++;
            else if (g.equals("B")) counts[1]++;
            else if (g.equals("C")) counts[2]++;
            else if (g.equals("D")) counts[3]++;
            else counts[4]++;
        }

        String[] labels = {"A", "B", "C", "D", "F"};
        System.out.println("\n=== 등급 분포 ===");
        for (int i = 0; i < 5; i++) {
            System.out.printf("%s: ", labels[i]);
            for (int j = 0; j < counts[i]; j++) {
                System.out.print("##");
            }
            System.out.println(" (" + counts[i] + "명)");
        }
    }

    // 과목별 히스토그램
    public static void printSubjectHistogram(ArrayList<Student> students, int subjectIdx) {
        String subjectName = students.get(0).getSubjects()[subjectIdx];
        int[] ranges = new int[5]; // 90+, 80-89, 70-79, 60-69, ~59

        for (int i = 0; i < students.size(); i++) {
            int score = students.get(i).getScores()[subjectIdx];
            if (score >= 90) ranges[0]++;
            else if (score >= 80) ranges[1]++;
            else if (score >= 70) ranges[2]++;
            else if (score >= 60) ranges[3]++;
            else ranges[4]++;
        }

        System.out.println("\n=== " + subjectName + " 점수 분포 ===");
        String[] labels = {"90+  ", "80-89", "70-79", "60-69", "~59  "};
        for (int i = 0; i < 5; i++) {
            System.out.print(labels[i] + " |");
            for (int j = 0; j < ranges[i]; j++) {
                System.out.print("**");
            }
            System.out.println(" (" + ranges[i] + ")");
        }
    }

    // 석차 정렬 (평균 기준 내림차순)
    public static ArrayList<Student> getRanking(ArrayList<Student> students) {
        // 복사본 만들기
        ArrayList<Student> ranked = new ArrayList<Student>();
        for (int i = 0; i < students.size(); i++) {
            ranked.add(students.get(i));
        }

        // 선택 정렬 (내림차순)
        for (int i = 0; i < ranked.size() - 1; i++) {
            int maxIdx = i;
            for (int j = i + 1; j < ranked.size(); j++) {
                if (ranked.get(j).getAverage() > ranked.get(maxIdx).getAverage()) {
                    maxIdx = j;
                }
            }
            // swap
            Student temp = ranked.get(i);
            ranked.set(i, ranked.get(maxIdx));
            ranked.set(maxIdx, temp);
        }

        return ranked;
    }

    // === Main ===
    public static void main(String[] args) {
        String[] subjects = {"국어", "영어", "수학", "과학", "사회"};

        ArrayList<Student> students = new ArrayList<Student>();
        students.add(new Student("김민준", subjects, new int[]{85, 92, 78, 90, 88}));
        students.add(new Student("이서연", subjects, new int[]{92, 88, 95, 87, 91}));
        students.add(new Student("박지호", subjects, new int[]{78, 72, 65, 80, 75}));
        students.add(new Student("최수아", subjects, new int[]{95, 98, 92, 94, 96}));
        students.add(new Student("정현우", subjects, new int[]{88, 85, 82, 79, 83}));
        students.add(new Student("강예린", subjects, new int[]{72, 68, 75, 70, 65}));
        students.add(new Student("조민서", subjects, new int[]{97, 95, 99, 96, 93}));
        students.add(new Student("윤하준", subjects, new int[]{63, 70, 58, 65, 72}));
        students.add(new Student("한소율", subjects, new int[]{81, 84, 87, 82, 80}));
        students.add(new Student("임도윤", subjects, new int[]{90, 91, 88, 93, 89}));

        // 1. 전체 성적표
        System.out.println("=== 전체 성적표 ===\n");
        printReport(students);

        // 2. 과목별 평균
        double[] subjectAvgs = getSubjectAverages(students);
        System.out.println("\n=== 과목별 평균 ===");
        for (int i = 0; i < subjects.length; i++) {
            System.out.printf("%s: %.1f\n", subjects[i], subjectAvgs[i]);
        }

        // 3. 1등
        Student top = findTopStudent(students);
        System.out.println("\n전체 1등: " + top.getName() + " (평균: " +
                           String.format("%.1f", top.getAverage()) + ")");

        // 4. 등급 분포
        printGradeDistribution(students);

        // 5. 수학 히스토그램
        printSubjectHistogram(students, 2); // 수학 = 인덱스 2

        // 6. 석차
        System.out.println("\n=== 석차 ===");
        ArrayList<Student> ranked = getRanking(students);
        for (int i = 0; i < ranked.size(); i++) {
            System.out.printf("%2d등: %-8s (평균: %.1f, 등급: %s)\n",
                    i + 1, ranked.get(i).getName(),
                    ranked.get(i).getAverage(), ranked.get(i).getGrade());
        }
    }
}
```

위 ExamAnalyzer에 다음 기능을 추가하세요:

1. **과목별 1등 출력** - 각 과목에서 가장 높은 점수를 받은 학생 이름과 점수
2. **특정 학생 상세 리포트** - 이름을 입력하면 해당 학생의 과목별 점수, 평균, 최고/최저 과목 출력
3. **과목 간 상관관계** - 두 과목을 선택하면 둘 다 80점 이상인 학생 수 출력

**완성 코드 (추가 메서드들):**

```java
import java.util.ArrayList;
import java.util.Scanner;

public class ExamAnalyzerExtended {

    static class Student {
        private String name;
        private int[] scores;
        private String[] subjects;

        public Student(String name, String[] subjects, int[] scores) {
            this.name = name;
            this.subjects = subjects;
            this.scores = scores;
        }

        public String getName() { return name; }
        public int[] getScores() { return scores; }
        public String[] getSubjects() { return subjects; }

        public double getAverage() {
            int sum = 0;
            for (int s : scores) sum += s;
            return (double) sum / scores.length;
        }
    }

    // 기능 1: 과목별 1등
    public static void printSubjectTopStudents(ArrayList<Student> students) {
        String[] subjects = students.get(0).getSubjects();
        System.out.println("\n=== 과목별 1등 ===");

        for (int s = 0; s < subjects.length; s++) {
            String topName = students.get(0).getName();
            int topScore = students.get(0).getScores()[s];

            for (int i = 1; i < students.size(); i++) {
                if (students.get(i).getScores()[s] > topScore) {
                    topScore = students.get(i).getScores()[s];
                    topName = students.get(i).getName();
                }
            }
            System.out.println(subjects[s] + ": " + topName + " (" + topScore + "점)");
        }
    }

    // 기능 2: 개인 상세 리포트
    public static void printStudentDetail(ArrayList<Student> students, String name) {
        Student found = null;
        for (int i = 0; i < students.size(); i++) {
            if (students.get(i).getName().equals(name)) {
                found = students.get(i);
                break;
            }
        }

        if (found == null) {
            System.out.println("'" + name + "' 학생을 찾을 수 없습니다.");
            return;
        }

        System.out.println("\n=== " + name + " 상세 리포트 ===");
        String[] subjects = found.getSubjects();
        int[] scores = found.getScores();

        int maxIdx = 0, minIdx = 0;
        for (int i = 1; i < scores.length; i++) {
            if (scores[i] > scores[maxIdx]) maxIdx = i;
            if (scores[i] < scores[minIdx]) minIdx = i;
        }

        for (int i = 0; i < subjects.length; i++) {
            System.out.printf("  %s: %d점", subjects[i], scores[i]);
            if (i == maxIdx) System.out.print(" (최고)");
            if (i == minIdx) System.out.print(" (최저)");
            System.out.println();
        }
        System.out.printf("  평균: %.1f점\n", found.getAverage());
        System.out.println("  최고 과목: " + subjects[maxIdx] + " (" + scores[maxIdx] + "점)");
        System.out.println("  최저 과목: " + subjects[minIdx] + " (" + scores[minIdx] + "점)");
    }

    // 기능 3: 과목 간 상관관계
    public static void printCorrelation(ArrayList<Student> students, int subj1, int subj2) {
        String[] subjects = students.get(0).getSubjects();
        int bothHigh = 0;

        for (int i = 0; i < students.size(); i++) {
            int s1 = students.get(i).getScores()[subj1];
            int s2 = students.get(i).getScores()[subj2];
            if (s1 >= 80 && s2 >= 80) {
                bothHigh++;
            }
        }

        System.out.println("\n=== " + subjects[subj1] + " & " + subjects[subj2] + " 상관관계 ===");
        System.out.println("둘 다 80점 이상: " + bothHigh + "명 / " + students.size() + "명");
        System.out.printf("비율: %.0f%%\n", (double) bothHigh / students.size() * 100);
    }

    public static void main(String[] args) {
        String[] subjects = {"국어", "영어", "수학", "과학", "사회"};

        ArrayList<Student> students = new ArrayList<Student>();
        students.add(new Student("김민준", subjects, new int[]{85, 92, 78, 90, 88}));
        students.add(new Student("이서연", subjects, new int[]{92, 88, 95, 87, 91}));
        students.add(new Student("박지호", subjects, new int[]{78, 72, 65, 80, 75}));
        students.add(new Student("최수아", subjects, new int[]{95, 98, 92, 94, 96}));
        students.add(new Student("정현우", subjects, new int[]{88, 85, 82, 79, 83}));
        students.add(new Student("강예린", subjects, new int[]{72, 68, 75, 70, 65}));
        students.add(new Student("조민서", subjects, new int[]{97, 95, 99, 96, 93}));
        students.add(new Student("윤하준", subjects, new int[]{63, 70, 58, 65, 72}));
        students.add(new Student("한소율", subjects, new int[]{81, 84, 87, 82, 80}));
        students.add(new Student("임도윤", subjects, new int[]{90, 91, 88, 93, 89}));

        // 기능 1: 과목별 1등
        printSubjectTopStudents(students);

        // 기능 2: 개인 상세 리포트
        printStudentDetail(students, "이서연");
        printStudentDetail(students, "윤하준");

        // 기능 3: 과목 간 상관관계
        printCorrelation(students, 0, 2); // 국어 & 수학
        printCorrelation(students, 1, 3); // 영어 & 과학
    }
}
```
