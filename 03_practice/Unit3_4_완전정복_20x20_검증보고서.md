## 0. 왜 v5를 다시 돌렸는가

사용자 본 의도: "**모든 내용이 반영된게 맞는지** 검증" — 즉 **콘텐츠 완전성**.

v1~v4가 본 것:
- EK 113개가 본문에 **언급**되는가 → 100% 매핑 확인 (표면)
- 사실 오류·산수·페이지 번호·QR 카운트 → CRIT 6 / MAJ 9 / MIN 23 발굴
- 하지만 **무작위 25개 샘플**만 깊이 검증 → 88개 EK는 미검증

v5가 본 것 (이게 본질):
- **EK 113개 전수 깊이 점수 (0~5점)**
  - 0 = 누락 / 1 = 언급만 / 2 = 정의만 / 3 = 정의+예시 / 4 = +트레이싱 / 5 = +"왜?"
- 4.5/4.10/4.13 알고리즘 서브불릿 (9+11+9=29개) 별도 점수
- 총 **142 EK 항목**의 깊이 점수표
