# AP CSA 모의시험 점수 변환표

> 모의시험 5세트 각각 채점 후 AP 점수를 예측하는 변환표입니다.
> 실제 커트라인은 매년 변동되며, 이 표는 2024-2025 공개 데이터 기준 추정치입니다.

---

## 채점 방법

### Section I: Multiple Choice (55%)

- 맞힌 문제 수 x 1점 = **MCQ 원점수** (최대 42점)
- **감점 없음** -- 모르면 무조건 찍기!
- 빈칸으로 두는 것보다 아무거나 찍는 게 항상 유리

### Section II: Free Response (45%)

각 문제별 배점:

| 문제 | 유형 | 배점 |
|------|------|------|
| Q1 | Methods and Control Structures | 0~7점 |
| Q2 | Class Design | 0~7점 |
| Q3 | Array/ArrayList | 0~5점 |
| Q4 | 2D Array | 0~6점 |
| **합계** | | **최대 25점** |

> FRQ는 **부분점수**가 있습니다. 완벽한 답을 못 써도 맞는 부분만큼 점수를 받습니다.

---

## 합산 공식 (비공식 추정치)

> **주의:** 아래 공식과 커트라인은 College Board가 공식 발표한 것이 **아닙니다**. 과거 시험 데이터와 커뮤니티 분석을 기반으로 한 추정치입니다. 실제 변환 공식은 매년 달라질 수 있습니다.

```
Composite Score = MCQ 원점수 x 1.0 + FRQ 원점수 x 1.8
```

| 항목 | 최대 원점수 | 가중치 | 최대 Composite |
|------|-----------|--------|---------------|
| MCQ | 42점 | x 1.0 | 42.0 |
| FRQ | 25점 | x 1.8 | 45.0 |
| **합계** | | | **87.0** |

## AP 점수 변환표 (비공식 추정치)

> **주의:** 아래 커트라인 점수는 College Board가 공식 공개한 것이 **아닙니다**. 과거 시험 결과와 커뮤니티 역추정을 종합한 근사치이며, 실제 커트라인은 해마다 다릅니다.

| Composite 범위 | AP 점수 | 의미 |
|---------------|---------|------|
| **62 이상** | **5** | Extremely well qualified |
| **47 ~ 61** | **4** | Well qualified |
| **33 ~ 46** | **3** | Qualified |
| **25 ~ 32** | **2** | Possibly qualified |
| **0 ~ 24** | **1** | No recommendation |

### 빠른 목표 계산

| 목표 점수 | MCQ 최소 (42문제 중) | FRQ 최소 (25점 중) | 대략적 정답률 |
|----------|---------------------|-------------------|-------------|
| **5점** | 30문제 + FRQ 18점 | 또는 35문제 + FRQ 15점 | ~70% 이상 |
| **4점** | 25문제 + FRQ 12점 | 또는 28문제 + FRQ 10점 | ~55% 이상 |
| **3점** | 20문제 + FRQ 7점 | 또는 22문제 + FRQ 6점 | ~40% 이상 |

> 위 조합은 예시입니다. MCQ를 많이 맞히면 FRQ 부담이 줄고, 반대도 마찬가지입니다.

## 약점 분석

### MCQ 약점 찾기

MCQ를 Unit별로 분류한 뒤 정답률을 계산하세요.

| Unit | 토픽 | 맞힌 수 / 전체 | 정답률 | 약점? |
|------|------|---------------|--------|------|
| Unit 1 | 변수, 타입, String, Math | ___/___  | ___% | |
| Unit 2 | Boolean, if/else, 반복문 | ___/___ | ___% | |
| Unit 3 | 클래스, OOP | ___/___ | ___% | |
| Unit 4 | Array, ArrayList, 2D Array | ___/___ | ___% | |
| 기타 | 정렬, 탐색, 재귀 | ___/___ | ___% | |

**정답률 60% 미만인 Unit이 약점입니다.** 해당 Unit의 강의노트를 다시 읽고, MCQ 문제집에서 그 Unit 문제를 추가로 풀으세요.

### FRQ 약점 찾기

| FRQ 유형 | Q번호 | 5세트 평균 점수 | 만점 | 달성률 | 약점? |
|----------|-------|---------------|------|--------|------|
| Methods & Control | Q1 | ___점 | 7점 | ___% | |
| Class Design | Q2 | ___점 | 7점 | ___% | |
| Array/ArrayList | Q3 | ___점 | 5점 | ___% | |
| 2D Array | Q4 | ___점 | 6점 | ___% | |

**만점 대비 50% 미만인 문제 유형이 약점입니다.**

### 약점별 처방

| 약점 | 처방 |
|------|------|
| Unit 1 (변수/String) | Unit1 강의노트 재독 + CodingBat String 문제 |
| Unit 2 (조건/반복) | 코드트레이싱 워크시트 집중 + De Morgan 드릴 |
| Unit 3 (OOP) | FRQ Q2 기출 반복 (2021~2025) |
| Unit 4 (배열) | 9가지 알고리즘 직접 코딩 + FRQ Q3, Q4 기출 반복 |
| FRQ Q1 낮음 | FRQ 풀이해설 참고 + 시험전략 가이드의 부분점수 전략 |
| FRQ Q2 낮음 | Unit3 강의노트 클래스 설계 파트 + 기출 Q2 반복 |
| FRQ Q3 낮음 | ArrayList 삭제 패턴 집중 연습 (i-- 또는 역순) |
| FRQ Q4 낮음 | 2D Array 순회 변형 연습 + 기출 Q4 반복 |

## 시험 당일 체크리스트

- [ ] Bluebook 앱 설치 및 로그인 확인
- [ ] 연필/펜 준비 (FRQ는 종이에 작성)
- [ ] 계산기 사용 불가 (가져가지 않아도 됨)
- [ ] 시험 시간: MCQ 90분 + 휴식 + FRQ 90분
- [ ] **MCQ 전략**: 모르면 무조건 찍기, 빈칸 남기지 않기
- [ ] **FRQ 전략**: Q1부터 시작, 부분점수 노리기, 컴파일 에러 최소화
