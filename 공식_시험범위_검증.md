### 4.2 4.6 Using Text Files — **`Scanner`는 File에서만 읽음**
- `File(String pathname)`
- `Scanner(File f)` — **`Scanner(System.in)`은 OUT** (1.4 EXCLUSION)
- `int nextInt()` / `double nextDouble()` / `boolean nextBoolean()`
- `String nextLine()` / `String next()`
- `boolean hasNext()`
- `void close()`
- 예외: `InputMismatchException` (타입 불일치), `IOException` 처리는 명시 없음 (try-catch 자체는 범위 밖이므로 단순 throws만 가능)

### 4.3 4.7 Wrapper Classes — **메서드 2개만**
- `Integer.MIN_VALUE`, `Integer.MAX_VALUE` (1.5에서 이미 등장)
- `static int Integer.parseInt(String s)`
- `static double Double.parseDouble(String s)`
- Autoboxing/unboxing은 자동 변환 (4.7.A.2/A.3)

### 4.4 4.8 ArrayList 메서드 — **6개만 출제 가능 (Java Quick Reference 명시)**
| 메서드 | 동작 |
|-------|------|
| `int size()` | 요소 개수 |
| `boolean add(E obj)` | 끝에 추가, true 반환 |
| `void add(int index, E obj)` | index 위치에 삽입 (`0 <= index <= size`) |
| `E get(int index)` | index 위치 요소 반환 |
| `E set(int index, E obj)` | 교체, **이전 요소 반환** |
| `E remove(int index)` | 제거, **제거된 요소 반환** |

> ⚠️ `contains()`, `indexOf()`, `isEmpty()`, `clear()`, `add(E)`의 boolean 활용 등은 **Quick Reference에 없음 → 출제 금지**

### 4.5 4.11 2D Array — **직사각형만**
- **EXCLUSION**: 비직사각형(jagged) 배열 OUT
- `arr.length` = 행 수, `arr[0].length` = 열 수
- `arr[row][col]` — **첫 인덱스가 행, 둘째가 열** (4.11.A.4)

### 4.6 4.14 검색 — **선형 검색만**
- 1D 배열, ArrayList, 2D 배열(행별 선형) 적용 가능
- **EXCLUSION (4.17.B.2)**: 선형/이진 외 검색 OUT
  - ⚠️ 이진 검색은 4.17에서 별도로 다룸 (재귀 단원)

### 4.7 4.15 정렬 — **선택/삽입 정렬만 (반복문 기반)**
- **Selection sort** (4.15.A.2): 미정렬 부분에서 최소/최대를 골라 swap
- **Insertion sort** (4.15.A.3): 미정렬 요소를 정렬 부분의 올바른 위치에 삽입
- 둘 다 **반복문(iterative) 기반**

### 4.8 4.16 Recursion — **추적만, 작성 불가**
- 재귀 메서드는 base case + recursive call 포함
- 각 호출은 자체 local variable/parameter
- 반복으로도 재귀로도 구현 가능
- 🔴 **EXCLUSION**: *"Writing recursive code is outside the scope of the AP Computer Science A course and exam."*
  - 의미: 학생은 주어진 재귀 메서드를 **트레이싱**(결과 추적, 호출 횟수 등)할 수 있음. **작성 요구 금지.**

### 4.9 4.17 Recursive Searching/Sorting
- **Binary search** (4.17.B): 정렬된 배열/ArrayList의 중간을 보고 절반 제거
  - 반복으로도 재귀로도 작성 가능 (4.17.B.3)
- **Merge sort** (4.17.C): 분할 정복 정렬 (재귀)
- **EXCLUSION**: 선형/이진 외 검색 OUT, selection/insertion/merge 외 정렬 OUT (즉 **bubble sort OUT**, quick sort OUT)
